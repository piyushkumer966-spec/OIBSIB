# OIBSIP - Java Development Tasks

Welcome to my Java Development internship repository for **Oasis Infobyte** (Spanning June 15, 2026 – July 15, 2026). This repository consolidates five core software engineering assignments built to demonstrate robust console-based application logic using **Java 8**.

---

## 🚀 Projects Overview

### 1. Online Reservation System
A secure reservation simulation mirroring commercial ticketing databases.
* **Key Features:** Account verification login gates, passenger details data capture, automated train name assignment, and transactional PNR lookup cancellation handlers.

### 2. Number Guessing Game
A round-based mathematical guessing game designed to track accuracy and scoring weight.
* **Key Features:** Automated random integer assignment (1–100), bounded attempts per round, multi-round session data mapping, and score scaling based on speed/attempts.

### 3. ATM Interface
A secure banking portal handling operational monetary tracking.
* **Key Features:** User ID and PIN verification, withdrawal constraint checking, instant deposit crediting, localized transaction history logging, and inter-account peer-to-peer balance transfer rules.

### 4. Online Examination
A web-inspired academic testing suite emphasizing runtime limitations.
* **Key Features:** Profile modification triggers, multi-choice question matrices, a strict 15-second countdown timer per question, and automatic blank submissions upon time expiration.

### 5. Digital Library Management
A comprehensive data directory splitting execution permissions across two clear operational roles.
* **Admin Module:** Privileged terminal for adding, modifying titles, and permanently deleting book records.
* **User Module:** Client desk supporting complete record lists, keyword-based search queries, real-time book check-out/return states, and a customer support log desk.
